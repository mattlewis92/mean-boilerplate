Congrats, you are now up and running!

This is a highly opinionated set of boilerplate code for starting off any kind of project utilising the MEAN stack (MongoDB, Express, AngularJS and Node.js).

Aims

  • Create a set of reusable components that are applicable to most projects (parts can easily be removed if you don't want them).
  • Create a logical and well thought out file structure that makes it easy to find the right piece of code.
  • Minimise require() on the backend as much as possible. So many node projects start off every file with a dozen lines of requiring other files - this is gross!
  • Minimise the task of writing boilerplate code to add new functionality to be as simple and enjoyable as possible.
  • Implement a truly stateless backend REST API - all templating is done via AngularJS and there are no sessions involved thanks to JWT.
  • Promises everywhere via bluebird. Having to use the async library for control flow feels a lot clunkier compared to just using promises.

Features

  • Automatic file watching, code linting and build process via Gulp.
  • Command line utility via commander.
  • Crons and background workers thanks to Kue and Node Cron
  • Angular inspired dependency injection but on the backend server thanks to Express DI and Dependable
  • Translation on both the frontend and backend thanks to i18n-2 and angular-translate.
  • Mongo ODM thanks to Mongoose
  • Server logging via Winston and frontend errors logged via exceptions.js
  • Frontend AngularJS routing powered by the ui-router
  • Styling via LESS and Bootstrap
  • Improved AngularJS controllers via angular classy
  • Frontend data store and model system powered by angular-data
  • And much more... (still to add)
